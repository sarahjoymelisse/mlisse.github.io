CSS Tooltip: They are user-triggered.

You may want additional details of the technical terms listed on A website. This information should only appear when the user has a technical term hovering over.

So here's the source code I learned as I developed some project and that CSS tooltip suddenly comes up in my mind

< style type="text/css">
/* Tooltip text */
.tooltip .tooltiptext {
visibility: hidden;
background-color:palegreen;
color: black;
text-align: center;
/* Position the tooltip text */
position: absolute;
}
/* Show the tooltip text when you mouse over the tooltip container */
.tooltip:hover .tooltiptext {
visibility: visible;
}
< /style>
