Quality Standard for Web Design

Web design is an important part of websites, design should help users understand your website and not to complicate things. It should be design in very organized way.

The most common use width on building web design as of now are: 1440px  | 1280px | 1024px.

I. Branding

Design will always start with branding and getting the correct colors to blend.

A. Logo

Company logo normally has 2 colors.  Some logos has 3 colors and the maximum colors that can be used are 4. If logo has one solid color, normally the second color would be black or white or grey. When you put the logo to the design, make sure its scale correctly and not pixelated.

B. Choosing Fonts

Fonts you use for website should be the same or near the font type of the logo. It will always be a part of branding. IF the fonts on the logo is not web fonts friendly, find a nearer or fonts similar on a web fonts library.

C. Website Design Color 

Always used the color of the logo when designing a website and start blending the font color, hover effects, links and background from there. Websites should only have 4 colors to blend as maximum. More than 4 is WRONG.

D. Background Solid Color

Background should always blend the logo. Logo should be clear enough to see when you use solid background. Normally logo has its background so better use it if you will use solid colors.

E. Background Image

Background image should be choose correctly. It will always be part of the branding. Make sure that if you used images, it should reflect what the company is all about. If there are no good image to use, you can use pattern backgrounds.

1. Full width web design. Never use full width background images on web design. Developer will have problem when it comes to auto resize of images when viewed to different screen size. And there will be no space to use image background.

2. Box style web design. Box style web design gives margin and space to contents and backgrounds. You can freely use background image or pattern image. 

II. Layout Sections

Website layout have basic guideline to follows. Below are the basic structures of the web design.

A. Header

Header is a placeholder for the ff.  Number 1 - 3 should always there and 4 - 7 are optional.

1. Logo
2. Search Bar
3. Menu navigation
4. Login section
5. Social network icons
6. Tagline
7. Email and contact details

B. Rotating Banner

Most common layout for web design is to have big banners, animated or static on every homepage. The purpose of this is to show what your websites is all about. OR if the website is ecommerce, mostly you need to put promotions or latest products.

C. Heading

Always add a Heading Title after the banner as a divider of your design. Always use 3 types of heading.  If you use the heading style, make sure all of it are the same on all pages.

H1 – The biggest font and bold letters – try to make 18px-21px maximum. Adding color is optional base on the logo.

H2 – Will always be a sub header – sub header normally contains summary of tagline supported by your header. Never bold, but can be italic in style. Use font size of 14-16 px.

H3 – Used only if you want a combination of presenting an article. Normally used for dates or comments. Font size should be 12 and normal style.

D. Home Page Contents

Design contents placement should be organized and neat to make it readable. Below are design guide when using columns.

1. One Column layout 

Use banner image or pleasant heading – make sure contents are long enough to make the website informative looking.

2. Two Column layout 

a. Use thumbnails images follow by H2 and contents, can make the heading color different for each column to show variations.

b. Can use icons + H2 followed by contents (make sure icons are related to the title)

c. Never use a and b at the same time.

d. Try to maximize columns into 3.

e. 4 columns will be use only if they have many contents to show.

E. Inner Page Contents

Inner page contents are very different from the homepage layout. Inner pages normally have fewer contents. So designer can make use this combination to make every pages informative.
1. Using Sidebar + Contents

Sidebar can be place left or right – if you use this in one inner page. Make sure all the inner pages will have this. Sidebar width should be around 240 - 300 px.

2. No Sidebar 

Without using sidebar make sure that you use the ff. layout.

a. Big Banner + 2 – 3 columns layout.

F. Sub Footer

Sub Footer has been on the trend now. It contains helpful information about the websites. Layout should be 3 - 4 columns to make use of this layout very nice. Designer can put the ff. on this section depends on what the contents the client can give.

1. About Us Summary
2. Newsletter signup
3. Our Services – bullet points
4. Latest News/Events
5. Social Network Account
6. Featured Gallery 
7. Contact Details
8. Location – Google Map
9. Useful Links / Menu navigator
10. Social network live feeds
11. Partners / Affiliates list
12. Client list

G. Footer

Footer fonts should be smaller than the content fonts. Contents that can be put here are the ff.

1. Copyright date and company name
2. Company logo + Copyright date
3. Copyright date and company name + social media icons
4. Powered by "company name" - for public facing site must be linked back " company's.com"

III. Dividers and Spaces

These are most of the common style in making the website readable and clean. Below are the things to take note when using each.

A. Dividers

Dividers is useful to section each content or information category. Below are types of dividers.

1. Thin Line ________________
2. Dashed line -------------------
3. Dotted line ……………………
4. Using different color per bg :

A. Spaces

Spaces should be the same on all pages. This should be explain to developer in detailed once the design is approved. It’s commonly included in the CSS. Always set 30px as minimum.

1. Spaces between logo and menu – 30px
2. Spaces between menu and big banner – 30 px
3. Spaces between big banner and Heading – 30px
4. Spaces between Heading and Contents – 30px
5. Spaces between Contents and Divider – 30px
6. Spaces between Divider and contents – 30px
7. Spaces between contents and sub footer – 30px

IV. Banners

Banners are important factors on web design. It can make your website a wow or a failure.

1. The 640 px height rule. Only see banner animation of first load. Make your banner size height to 600px as maximum.

2. Banner + Important contents. Banner height size should be 400 - 550px maximum to show some important contents without user scrolling it.

3. Use PNG or JPEG format.

V. Thumbnails

Thumbnails are mostly generated, but designers can control this on the layout to make It more neater.

1. Border Size. Border size should be 1px min up to 4px maximum. Don’t make border to thick. You can add color if you want it to stand out.

1. Thumbnail Size. Thumbnails should always be square size for responsive layout also. Size should be 100 x 100| 120 x 120| 150 x 150| 180 x 180| 200 x 200| 220 x 220| 250 x 250| maximum.

3. Use PNG or JPEG format.
